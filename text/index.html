  1. h1
    is used for main headings
  2. h2

  3. h3 to h6

HTML formatting elements

Formatting elements were designed to display special types of text:

  1. b
    by using b element we can make characters appear bold
  2. i
    by using i element we can make characters appear italic
  3. u
    by using u element we can make characters appear underline.

superscript & subscript:

Superscript and subscript elements are used to show text slightly above or below the normal line for scientific and mathematical expressions:

  1. sup
    sup element is used to display text slightly above the normal line, Example: H2
  2. sub
    sub element is used to display text slightly below the normal line, Example: H 2

Semantic Markup

Semantic markup is used to clearly describe the meaning of content to the browser:

  1. strong
    element is used to show that the text is very important and Browsers usually make it bold, Example: This toy has many small pieces and is not suitable for children under five years old.
  2. em
    element is used to emphasis to the text (like stress in speech) and Browsers usually make it italic, Example: This is HTML website.

Quotations

exactly from someone else’s words:
(There are two elements commonly used for marking up quotations:)

  1. blockquote
    blockquote element is used for longer quotes that take up an entire paragraph (cite attribute is usally used with this element), Example:

    Did you ever stop to think, and forget to start again?

  2. q
    element is used for short quotations, inside a sentence., Example: She said, Knowledge is power.
    (Browsers usually display it with quotation marks.)

    Note: The q and blockquote elements may use the cite attribute to specify the source of the quotation. The value of the cite attribute should be a URL pointing to a webpage or document that gives more information about the quote’s origin.

Changes to Content

Inserted, deleted and strikethrough elements help show changes in content. ins is for added text, del for removed text, and s for outdated information:

  1. ins
    element is used to show text that has been removed from a document because it is incorrect or replaced after editing. Browsers normally display inserted text with an underline to highlight the change.

    Example: This is updated text.

  2. del
    element is used to show text that has been removed or is no longer needed in the content. It appears with a strikethrough, indicating that the text is deleted.

    Example: This is incorrect information.

  3. s
    element shows text that is no longer accurate, valid, or relevant, but still kept visible for reference — like old prices or outdated info. It also displays with a strikethrough, but not for document editing purposes.

    Example: Old price: $100 $75
